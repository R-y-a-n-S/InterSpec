InterSpec is licensed under the LGPL v2.1 license, and is built using a number of additional open-source libraries.

This file provides a description of libraries used, as well as their license text.
Library versions used are current as of InterSpec v1.0.13.

  • Wt (version 3.7.1): a C++ library for web development.
    InterSpec uses Wt extensively to render the HTML based GUI, to facilitate user interactions, and for database object-relational mapping.
    Wt is available under either the GNU GPL Version 2 license, or a commercial license; development of InterSpec has been done under the commercial license.
  • Boost (version 1.79): C++ libraries.
    InterSpec uses Boost for a host of tasks, including: as a dependency to Wt, unit testing, mathematical constants and distributions, input parsing, working with date/times, linear algebra, hashing, supplements to the c++ language, and more. Boost is licensed under the Boost Software License.
  • RapidXml (version 1.13): C++ XML Parser.
    A slightly modified version of this library, which allows parsing of some non-compliant XML files, is used for all XML parsing tasks within InterSpec; all modifications are included and marked with the source code of InterSpec. RapidXml is licensed under the Boost Software License.
  • pugixml (version 1.9): Light-weight, simple and fast XML parser for C++ with XPath support.
    pugixml is used by the spectrum file query tool to parse an XPath query event XML files. pugixml is licensed under the MIT License. (Windows, Linux, macOS builds only)
  • Minuit2 (version 5.28.00): Minimization Package.
    Minuit2 is part Cern's ROOT package, and is used to perform numerical optimizations for peak, energy calibration, and shielding/source-activity fitting. Minuit2 is licensed under the LGPL v2.1 License.
  • Cuba (version 3.0): a library for multidimensional numerical integration
    Cuba is used to perform the numericall integration for self-attenuation sources in the shielding/source tool. Cuba is licensed under the LGPL v3 License.
  • muparserx (version 4.0.7): c++ Math Parser Library.
    muparserx is used to evaluate user entered detector efficiencies, and to help evaluate expressions for the TerminalWidget (not all builds of InterSpec contain this tool). muparserx is licensed under the BSD License.
  • Electron (version 17.4.11): Package based on the Chromium web browser to build cross platform desktop apps with JavaScript, HTML, and CSS.
    Electron is used to display app contents for the Electron-based Windows and Linux versions of InterSpec. Electron is licensed under the MIT License, but libraries it uses may be licensed under other terms; see Electrons git repository for more information.
  • Microsoft.Web.WebView2 (version 1.0.705.50): Embed web content (HTML, CSS, and JavaScript) in your native applications with Microsoft Edge WebView2.
    Microsoft.Web.WebView2 is used to display app contents for the "Edge" version of InterSpec on Windows (starting from InterSpec version 1.0.11). Microsoft.Web.WebView2 is licensed under the Microsoft software licensing terms.
  • wxWidgets (version 3.2.1): Cross-Platform GUI Library.
    wxWidgets is used by the "Edge" version of InterSpec on Windows to handle application-level events like GUI window creation. wxWidgets is licensed under the wxWindows Library Licence, Version 3.1.
  • SQLite (version 3.17.0): SQL database engine.
    SQLite is used to save user preferences and application state. The version used is what is included in the Wt library. SQLite is Public Domain software.
  • ROOT: A modular scientific software framework.
    InterSpec uses a small amount of code derived from the ROOT TSpectrum class to estimate the continuum for a spectrum (both are deprecated features); code used is marked and attributed within the InterSpec source code. ROOT is licensed under the LGPL v2.1 License.
  • jQuery (version 3.6.0): javascript library.
    jQuery is used to help deal with cross-browser idiosyncrasies in the client-side java script. jQuery is licensed under the MIT License.
  • qTip2 (version 3.0.3): javascript library that provides tooltip and notification tools.
    qTip2 is used to help make the content-rich tooltips, as well as the notifications within InterSpec. qTip2 is licensed under the MIT License.
  • d3.js (version 3.5.15): a JavaScript charting library.
    d3.js powers rendering and interactions of the spectrum chart, gross-count time chart, and decay tool. d3.js is licensed under the BSD License.
  • jQuery QueryBuilder (version 2.5.2): jQuery plugin for user friendly query/filter creator.
    QueryBuilder is used to construct spectrum file queries. QueryBuilder is MIT licensed javascript library. (Windows, Linux, macOS builds only)
  • spectrum.js (version 1.8.0): The No Hassle jQuery Colorpicker.
    spectrum.js is select colors throughout InterSpec when a native HTML5 is not used. spectrum.js is MIT licensed javascript library.
  • resize-observer-polyfill (version 1.5.1): A polyfill for the Resize Observer API.
    resize-observer-polyfill is used to support the JavaScript ResizeObserver functionality for older browsers that do not natively support API. resize-observer-polyfill is MIT licensed javascript library.
  • zlib (version 1.2.12): A Massively Spiffy Yet Delicately Unobtrusive Compression Library.
    zlib is used to open user uploaded zip files that contain spectrum files. zlib is ZLIB licensed library.
  • libHaru (version 2.3.0): free PDF library.
    libHaru is used for the macOS build to help provide the Quick Look preview of spectrum files inside the Finder. libHaru is ZLIB licensed library.
  • QR Code generator library (version 1.8.0): Multiple programming language QR code generator.
    QR Code generator library is used to generate QR codes throughout InterSpec; notably to represent detector response functions. QR Code generator library is MIT licensed library.
  • libpng (version 1.6.37): libpng is the official PNG reference library.
    libpng is only on the macOS app build as part of rendering spectrum file previews in the Finder. libpng is PNG Reference Library License version 2 library.
  • Ceres Solver (version 2.1.0): Ceres Solver is an open source C++ library for modeling and solving large, complicated optimization problems.
    Ceres Solver is used, starting with InterSpec version 1.0.11, for the the Relative Efficiency tools ("Isotopics by nuclides" and "Isotopics from peaks"); it may be used for peak-fitting as well in the future. Ceres Solver is BSD License library.
  • Eigen (version 3.4.0): Eigen is a C++ template library for linear algebra: matrices, vectors, numerical solvers, and related algorithms.
    Eigen is used used by the Ceres Solver library, as well as a few places within the Relative Efficiency tools for some matrix operations. Eigen is Mozilla Public License Version 2.0 library.
